## Installation

**After installation a recommendation is to also use the Jetbrains Mono Font with the following settings:**

```
{
  "editor.fontFamily": "Jetbrains Mono, Monaco, 'Courier New', monospace",
  "editor.fontSize": 13,
  "editor.lineHeight": 1.4,
}
```
